## 第三部分：AI动作识别（核心演示）(15分钟)

### 1. 准备视频 📹
**需要准备：**
- 一段排球动作视频（垫球/扣球/发球等）
- 建议时长：5-30秒
- 格式：MP4（推荐）
- 大小：<50MB
- 要求：人物清晰、完整动作

### 2. 进入AI教练 🤖

点击 **"AI 教练"** 标签

点击 **"开始咨询 AI 教练"** 按钮

### 3. 上传视频 📤

**方式A: 拖拽上传**
- 将视频文件拖到上传区域
- 松手即可

**方式B: 点击上传**
- 点击上传区域
- 选择视频文件
- 确认

### 4. 视频预览 👀
- 查看视频预览
- 确认视频正确

### 5. 选择分析模式 🎯

**推荐新手：单帧快速分析**
- ⚡ 速度快（2-5秒）
- 📊 提取关键帧分析
- 💡 适合快速反馈

**进阶用户：连续帧深度分析**
- 🔬 更详细（10-30秒）
- 📈 分析整个动作序列
- 🎯 包含轨迹分析

### 6. 开始AI分析 🚀

点击 **"开始AI分析"** 按钮

**你会看到：**
- 🔄 加载动画（Spinner）
- 💬 "AI正在分析中..."

**等待2-30秒（取决于模式）**

### 7. 查看分析结果 📊

#### 总分卡片
- 🏆 总分：85.5分
- ⭐ 等级：高级
- 📈 巨大的分数显示

#### 四项分项得分
```
手臂动作: 90.0分 🔵 蓝色进度条
身体姿态: 85.0分 🟢 绿色进度条
位置准确: 82.0分 🟣 紫色进度条
稳定性: 85.0分 🟠 橙色进度条
```

#### 姿态检测可视化图 🎨
- 显示你的姿态
- 标注关键点
- 连接骨架线

#### 运动轨迹分析图 📈（序列模式）
- 显示动作轨迹
- 颜色编码时间轴
- 关键点运动路径

#### AI反馈建议 💡
```
例如：
✓ 手臂伸直程度很好
✓ 重心稍微前移可以更稳定
✓ 整体动作流畅度优秀
```

### 8. 操作选项 🎮
- **关闭**: 返回主页
- **再次分析**: 上传新视频

## 📞 演示中可能遇到的问题

### Q: 视频分析时间太长？
A: 选择"单帧快速分析"模式，2-5秒即可完成

### Q: 评分为什么不高？
A: 确保视频中：
- 人物姿态清晰
- 光线充足
- 动作完整
- 画面稳定

### Q: 某个功能点不到？
A: 可能是：
- 需要达到一定等级/星星
- 功能被锁定（解锁提示会显示）

### Q: 页面显示异常？
A: 
- 刷新浏览器（F5）
- 清除缓存（Ctrl+Shift+Del）
- 更换浏览器
