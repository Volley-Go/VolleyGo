# 🏐 排球冒险 - 快速启动指南（Web版本）

## 🎯 一分钟快速启动

### 步骤1: 安装依赖
```bash
pip install -r requirements.txt
```

### 步骤2: 启动Web服务器
```bash
# Windows用户
run_flask.bat

# Linux/Mac用户
chmod +x run_flask.sh && ./run_flask.sh
```

### 步骤3: 打开浏览器
访问: **http://localhost:5000**

就这么简单！🎉

## 🐛 遇到问题？

### 常见问题快速解决

**问题1: 视频上传失败**
```
解决: 检查文件格式（MP4/AVI/MOV/MKV）和大小（<50MB）
```

**问题2: API连接失败**
```
解决: 
1. 确认Flask服务器正在运行
2. 检查端口5000是否被占用
3. 查看浏览器Console错误
```

**问题3: 分析结果不准确**
```
解决:
1. 确保视频中人物姿态清晰
2. 尝试不同的分析模式
3. 调整视频拍摄角度
```

**问题4: 页面显示异常**
```
解决:
1. 清除浏览器缓存（Ctrl+Shift+Del）
2. 使用Chrome/Edge/Firefox最新版本
3. 检查网络连接（Tailwind CSS使用CDN）
```
